Crema To-Do (React)

A simple to-do app built using React and tailwindcss.

What is Crema To-Do?

Crema To-Do (React) is a simple to-do app built using React and tailwindcss. In this application you can add, edit or delete to-do. However, to-do filtering, searching, etc. You can perform operations and make adjustments from the preferences section.

What other features does Crema To-Do have?

You can sort the to-dos ascending-descending or by their names-dates, set whether to-dos show the date or time information under which the to-do was added, and when you accidentally delete a to-do (only the last to-do you deleted and within 7 seconds) you can undo.
Important: This repo is a version of the same project written in React. For the same project written in Vue, check out the Crema To-Do (Vue) page. The Vue version of the same project was first written, and a year later the same project was adapted to React.

What is the technology stack of Crema To-Do?

React and tailwindcss.

What is the story behind Crema To-Do?

To be honest, there is no story behind it. I just wanted to try Vue.js and tailwindcss. Later, I decided to rewrite the same project using React.

Do you have any plans for Crema To-Do?

Although it is not certain, I can rewrite this application to use the database instead of the browser local storage. I have no idea whether I can write in Vue or React.
